## 開発情報

- `git clone https://github.com/watagori/discord-bot-ts-titech`
- `cd discord.ts-example`
- `npm install`
- `npm run build`
- `set BOT_TOKEN=<your bot token>`

  token に関することがわからない場合はこちら [discord developer portal](https://discord.com/developers/)

- `npm run start`

## ブランチ関連

| ブランチ名    | 目的     | 注意                           |
| ------------- | -------- | ------------------------------ |
